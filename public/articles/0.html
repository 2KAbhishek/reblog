New Blog

I made a new blog while exploring React and Wordpress and these are my findings.

Advantages

  • Uses React with Webpack and genrates optimized bundles for deployment.
  • Minimal and clean design, puts all focus on the blog content.
  • Easy to add new articles using Wordpress.
  • Has a completely responsive design, works everywhere.

    • Disadvantages

  • No server side rendering
  • No support for Markdown, which would make articles easier to write
  • External dependency on Wordpress for writing articles

    • Improving

    Using something like Next.js could help tackle the SSR problem, but seems too much for a blog.
    Static site generators like Gatsby, Jekyll or Hugo can help solve a lot of the drawbacks.
